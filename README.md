# Event Management Platform

## Overview

Event Management Platform is a modern web application designed to simplify event planning, organization, and management. The platform provides an intuitive interface for managing events, schedules, participants, and event-related activities efficiently.

## Features

* Event creation and management
* Interactive dashboard
* User-friendly interface
* Responsive design
* Real-time event organization
* Modern UI built with React and Tailwind CSS
* Fast performance with Vite

## Technologies Used

* React
* TypeScript
* Vite
* Tailwind CSS
* JavaScript
* HTML5
* CSS3

## Installation

Clone the repository:

```bash
git clone https://github.com/Shivapriya411/event-management-platform.git
```

Navigate to the project folder:

```bash
cd event-management-platform
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:5173
```
